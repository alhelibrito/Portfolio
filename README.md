### Proyectos Destacados

#### 1. Web Scrapping para Datos Climáticos

* **Descripción:** Este proyecto demuestra cómo extraer datos directamente de una página web utilizando Python. Se enfoca en obtener información climática (o los datos específicos que hayas extraído) y prepararla para un análisis posterior.
* **Habilidades Demostradas:** Web Scrapping (con librerías como `BeautifulSoup` y `requests`), manejo de datos con `Pandas`, y guardado de datos.
* **Herramientas:** Python, `BeautifulSoup`, `requests`, `Pandas`, Google Colab.
